# Code of Conduct

## Our Commitment

We want this project to be a welcoming, safe, and professional space for everyone.
We commit to respectful collaboration across experience levels, identities, and backgrounds.

## Expected Behavior

- Be respectful and constructive in issues, pull requests, and discussions.
- Assume good intent and ask clarifying questions before escalating.
- Give and receive feedback focused on code and outcomes, not people.
- Share context when reporting bugs, incidents, or regressions.
- Prioritize safety and integrity when discussing trading behavior and risk.

## Unacceptable Behavior

- Harassment, hate speech, discriminatory language, or personal attacks.
- Trolling, intimidation, or repeated disruptive behavior.
- Sharing private data, credentials, or sensitive account information.
- Misleading technical claims made in bad faith.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, contact maintainers through a private channel or open a private security/advisory report when appropriate.

Include:

- What happened
- Where it happened (link to issue/PR/discussion)
- Approximate time
- Any supporting context

## Enforcement

Maintainers may take action based on severity and pattern, including:

- Requesting changes in behavior
- Content removal or moderation
- Temporary or permanent ban from project spaces

Enforcement decisions are made to protect the community and project health.

## Scope

This Code applies in all repository spaces, including issues, pull requests, discussions, and review comments.
